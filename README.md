# Portfólio Premium de Dados e BI

Landing page front-end para um portfólio profissional de Dados e BI, criada com Vite, React, JavaScript, Tailwind CSS, Framer Motion e Lucide React.

## Tecnologias

- Vite
- React
- JavaScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Como rodar

```bash
npm install
npm run dev
```

O projeto ficará disponível no endereço exibido pelo Vite, normalmente:

```bash
http://127.0.0.1:5173
```

## Build de produção

```bash
npm run build
```

## Estrutura principal

```text
src/
  components/
  data/
  App.jsx
  index.css
  main.jsx
```

Os textos, links, projetos, áreas de atuação e diferenciais ficam centralizados em `src/data/profileMock.js`.
