## Sobre mim

Atuo em Dados e BI com uma base técnica que passa por banco de dados, APIs, automações e interfaces analíticas. Meu trabalho costuma começar antes do dashboard: organizo a estrutura da informação, entendo a regra de negócio, preparo a camada de acesso e entrego uma experiência analítica que faça sentido para a operação.

Tenho experiência em contextos de saúde, logística, estoques, compras, performance operacional e produtos internos orientados por dados. Já trabalhei com bases públicas e privadas, construção de dashboards corporativos, modelos de análise, pipelines, integrações, módulos de BI para aplicações web e backend próprio em Python com FastAPI.

Gosto de projetos em que o dado precisa virar estrutura, não apenas visualização. Para mim, análise boa não nasce no gráfico. Nasce na forma como a informação foi organizada, validada e disponibilizada para quem precisa decidir.
